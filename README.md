## Setup .env

Create `.env` file, then insert your mongodb string into the file

```env
MONGODB_URI = XXX
```

## Installation

```bash
# use yarn
$ yarn install

# or npm
$ npm install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

<p align="center">Open Browser, go to url <a href="http://localhost:3000/" target="_blank">localhost:3000</a>, then you will see start page</p>
